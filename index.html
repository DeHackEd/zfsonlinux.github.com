General

The native Linux kernel port of SUN's ZFS filesystem. To get started with ZFS on Linux simply download the latest release, then follow either the rpm or deb package instructions to build and install packages for your system. Alternately, there is a ppa available for Ubuntu users.

ZFS Core Features
Storage pools Massive Capacity Copy-on-write transactions
Clones Dynamic striping Variable block sizes
Snapshots Deduplication Online Repair
Lightweight FS creation Cache management Adaptive endianness
